# rsa_tool/README.md
# RSA Cryptography Tool - Web Interface

Web-based tool for demonstrating RSA cryptographic operations based on CLRS Chapter 31.

## Project Structure
rsa_tool/
├── app.py # Main application entry point
├── config/ # Configuration
│ └── settings.py
├── controllers/ # Route handlers (MVC Controllers)
│ ├── key_controller.py
│ ├── crypto_controller.py
│ ├── demo_controller.py
│ └── view_controller.py
├── services/ # Business logic (MVC Model)
│ ├── rsa_service.py
│ ├── demo_service.py
│ └── utils_service.py
├── models/ # Data models
│ └── response_model.py
├── views/ # HTML templates (MVC View)
│ ├── layouts/
│ ├── components/
│ └── pages/
└── static/ # CSS, JS, Images
├── css/
├── js/
└── images/


## Installation

1. Install dependencies:
```bash
cd d:\projects\Project-1
pip install -r requirements.txt
```
